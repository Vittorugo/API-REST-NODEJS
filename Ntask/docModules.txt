Módulos/Frameworks instalados para criar esta aplicação ...


Babel - Módulo que vai auxiliar na utilização das novas funcionalidades do ES6/7.

    npm install --save-dev babel-cli

Express -

    npm install express --save

Sequelize - Framework para BD 

    npm install sequelize --save
    npm install -D sequelize-cli 


Mysql - BD 

    npm install mysql --save

Body Parser - middleware para as requisições

    npm install body-parser --save
